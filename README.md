# FocusFlow AI Landing Page

A modern, responsive landing page for an AI productivity assistant, built with HTML and Tailwind CSS.

## Overview

This is a clean, single-page website designed to market "FocusFlow AI", a tool for deep work and focus management. It features a minimal aesthetic with a light theme, consistent with modern SaaS design trends.

## Features

- **Hero Section:** Bold value proposition with a split CTA and an abstract UI mockup.
- **Problem Statement:** "Why" section addressing distraction and productivity loss.
- **Features Grid:** 3 key benefits using a clean card layout.
- **Process Steps:** "How it Works" section with visual cues.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
- **Tech Stack:** Pure HTML5 + Tailwind CSS (via CDN for simplicity).

## Development

Since this project uses the Tailwind CSS CDN, no build step is strictly required for development. You can simply open `index.html` in any browser.

To serve locally with a live server:

```bash
npx serve .
```
